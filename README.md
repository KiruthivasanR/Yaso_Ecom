# Yasodha Dairy – Promotional Website

Modern glassmorphism React landing page for a premium dairy brand, built with Vite, Framer Motion, and responsive CSS.

## Tech Stack

- React (Vite)
- Framer Motion for animations
- Vanilla CSS (no Tailwind)

## Development

```bash
npm install
npm run dev
```

## Notes

- All hero and product imagery currently uses high-quality remote images from Pexels/Unsplash. You can replace them with your own 4K assets by updating the `background-image` URLs in `src/styles/main.css` or by adding local images under `src/assets/images` and pointing the CSS to those paths.
